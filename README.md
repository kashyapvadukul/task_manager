# task_manager
This Task Manager Application is a lightweight Python console tool for organizing daily tasks. It allows users to add tasks with descriptions, due dates, and priorities, as well as update, delete, and view them. Tasks are stored in JSON format with input validation for a smooth experience.

# Task Manager (Console)
A simple menu-driven Task Manager written in Python.

## Features
- View, Add, Update, Delete tasks
- Tasks persisted to `tasks.json`
- Input validation and simple confirmations

## Run
```bash
python main.py
```

## Files
- `main.py` - Main application script
- `tasks.json` - Data storage (JSON).
- `README.md` - Project info
